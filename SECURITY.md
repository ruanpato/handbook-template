# Security Policy

This repository is an **educational handbook** (Markdown content), not a running service.

## Scope & ethics

Any offensive or hazardous technique is **defensive/educational**, framed as **authorized**, and
reproducible only in **labs, CTFs, your own systems, or explicitly authorized targets**. The
handbook does not provide functional malware, real targets, real data, or operationalized abuse.

## Reporting a problem

If you find leaked secrets, personal data committed by mistake, operationalizable content, or a
malicious link/dependency, report it **privately**:

- Open a [GitHub Security Advisory](../../security/advisories/new), or
- Contact the maintainer via their GitHub profile.

We follow **coordinated disclosure**: private report → fix → public note.
