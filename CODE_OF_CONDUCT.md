# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/), v2.1.

Be respectful, assume good faith, and keep discussion constructive. Unacceptable behavior includes
harassment, personal attacks, and publishing others' private information.

**Reporting:** open a private [security advisory](../../security/advisories/new) or contact the
maintainer via their GitHub profile. Reports are handled confidentially.
